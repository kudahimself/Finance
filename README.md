# Quant Trading Research Project

This project is designed for researching, backtesting, and visualizing quantitative trading strategies.

## Structure
- `data/` — Data loading and management
- `strategies/` — Trading strategy implementations
- `backtesting/` — Backtesting engine and scripts
- `visualization/` — Visualization tools and scripts
- `utils/` — Utility functions

## Getting Started
1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Launch Jupyter Notebook:
   ```sh
   jupyter notebook
   ```
3. Explore and run the sample notebook in the project root.

## Notes
- Add your own data to the `data/` folder.
- Implement new strategies in `strategies/`.
- Use `backtesting/` for running and evaluating strategies.
- Visualize results with scripts in `visualization/`.
